# Hyas ⚡️

> Hugo-powered Jamstack framework

- Hugo-powered
- Jamstack
- Community
- Security aware
- Fast by default
- SEO-ready
- Development tools
- Bootstrap framework
- Netlify-ready

## Prerequisites

Hyas requires [Node.js](https://nodejs.org/) v16.17.0 or higher.

## Create a new project

With NPM:

```bash
npm create hyas@latest
```

With Yarn:

```bash
npm create hyas
```

With PNPM:

```bash
pnpm create hyas
```

Then follow the prompts!
